## 使用说明

**搜索题目** :

- 按下 `F1` 或 `Ctrl+Shift+P` ，查找`Luogu: 搜索题目`命令，点击后输入题号完成搜索

- 点击界面左侧洛谷图片，然后点击搜索图标，输入题号完成搜索

**提交题目** :

- 按下 `F1` 或 `Ctrl+Shift+P` ，查找`Luogu: 提交当前文档代码`命令，点击后按照步骤进行即可

其他功能正在开发，尽情期待
